軽量な 2D 作図アプリ。

レイヤー、寸法、複数ページの投影図に対応します。PDF や SVG への出力もできます。必要なら、図面から 3D プレビューや STL 出力も。

デスクトップ版をダウンロード ブラウザで試す ドキュメント

デスクトップ版は AI 連携（MCP）・テキストの 3D 輪郭化に対応。初回リリースは GitHub Releases から（未署名・無料）。

2D 作図

図形、レイヤー、寸法線。Undo 付きの SVG キャンバス。

複数ページ

上面図・正面図など、ページごとに管理。

3D プレビュー

投影図が揃えば立体化。STL 出力も可能。

AI 連携（デスクトップ）

Claude Desktop などから MCP 経由で図面操作。ブラウザ版では利用できません。
