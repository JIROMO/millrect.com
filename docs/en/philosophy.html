Documentation / Philosophy

Philosophy

Millrect is a lightweight drawing-first CAD tool. 3D, AI, and export features are designed as supporting layers that grow from the drawing, not as replacements for it.

On this page
  • Drawings come first
  • 3D is derived
  • Small and inspectable
  • AI assists rather than takes over
  • Portable by default

Drawings come first

In Millrect, the first object you work with is not a 3D mesh. It is a 2D drawing: top views, front views, dimensions, shapes, and pages. Keeping intent in the drawing makes the work easier to explain and easier to return to later.

3D preview and STL export are useful, but the editable source remains the drawing. Update the drawing, then regenerate the solid from it.

3D is derived

Millrect builds 3D from multiple orthographic views, such as top and front projections. Instead of directly sculpting a mesh, it keeps the 2D profiles that justify the result.

IdeaHow Millrect treats it
Editable source2D drawings, pages, dimensions, and contours
Generated result3D preview and STL derived from drawings
IterationEdit the drawing, then regenerate

Small and inspectable

Millrect keeps drawing, previewing, and saving connected to the same project data. The browser and desktop app work with the same files, so the tool stays approachable while still being useful for real output.

When features are added, the priorities are predictable drawing behavior, readable project files, and Undo / Redo that matches what users expect.

AI assists rather than takes over

MCP integration and sketch import are meant to reduce setup friction. The default flow is proposal, inspection, adjustment, then confirmation on the canvas.

Automation you can inspect

AI integration is not a replacement for the drawing. It is scaffolding for creating one, with the final shape left visible and editable on the canvas.

Portable by default

Projects can be saved as JSON, and outputs can be exported as PDF, SVG, or STL depending on the job. The goal is to keep work openable, shareable, and usable outside a single screen.
