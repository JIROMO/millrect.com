2D Drawing

Draw shapes from the left tool palette and add dimensions and annotations. Closed outlines become the material for 3D generation.

Tools

ToolKeyUsable for 3DSummary
SelectV—Select, move, and resize shapes
PanH—Move the canvas
RectangleR✓Rectangle (rounded corners optional)
CircleC✓Circle
LineL—Construction lines, center lines
Pen—✓ (when closed)Bezier curves and free outlines
TextT—Notes and labels
DimensionD—Length display (annotation)

Only closed outlines can be used for 3D. Lines, text, dimensions, and open beziers are not included in 3D generation.

The fill color of rectangles, circles, and similar shapes affects both the 2D display and the 3D preview mesh color. See Editing — Appearance (fill & stroke) for how to set colors.

Rectangle (R)

  1. Select the rectangle tool and drag on the canvas
  2. Hold Shift while dragging to draw a square
  3. After selection, edit X/Y/W/H, fill and stroke, and corner radius (rx) in the Design tab

Circle (C)

  1. Select the circle tool and drag from near the center to set the radius
  2. After selection, edit center (cx, cy) and radius (r) in the Design tab
  3. Circles become cylindrical or hole outlines in 3D generation

Line (L)

  1. Click (or drag) start and end points with the line tool
  2. Use for construction lines, center lines, reference lines, and cut (score) lines
  3. Does not affect the 3D solid, but works as a snap reference

Lines can be distinguished by their role. Select a line and switch it under “Line role” in the Design tab; the display style changes per role. The default is Drawing.

RoleUseDisplay3D
DrawingGeneral drawing / reference lines (default)SolidNo effect
Cut / scoreCut or slit intent (annotation)Solid (distinguished by color)No effect
AnnotationLeader lines, callouts, note helpersShort dashesNo effect
ConstructionTemporary guides / layout linesDash-dot, semi-transparentNo effect
Cuts live in the shape geometry itself. Line role is display style only; a role: "cut" line never implicitly removes 3D material. To make a cut show up in 3D, put the notch into the shape's own geometry — e.g. boolean-subtract a rectangle/circle, or include the slit in a closed path outline. Then what you see in 2D is exactly what you get in 3D (derived). Module Joint 1's keyholes (hole + slit to the edge) are baked into the outline path this way.

Pen (Bezier)

  • Click to add points; drag to adjust curve handles
  • Click the start point again to close the path (becomes a 3D outline)
  • While drawing: Enter to confirm, Backspace to remove the last point
  • Select a shape with the select tool and use path edit mode to edit nodes individually

Text (T)

  1. Click with the text tool and enter a string (an edit dialog opens)
  2. Change content and formatting in the Design tab, or right-click → Edit text
  3. Use for part names, material names, and other notes

Main properties for text shapes:

Display preview and outline conversion use the HarfBuzz text engine, rendering glyph outlines from registered font TTF bytes. The same engine works in a browser when served over HTTP (npm run serve).

Using text in 3D

Text shapes themselves are not 3D outlines. Convert to outline to turn them into path shapes usable for 3D generation.

Project fonts (Google Fonts)

From Project fonts (optional) in the right panel Pages tab, you can add Google Fonts to the project. In addition to bundled Gen Interface JP, choose typefaces for logos and notes.

Not fonts installed on your PC

The Find fonts… modal lists the Google Fonts catalog via the Fontsource API (online). It is unrelated to OS font folders or queryLocalFonts. The catalog is fetched from Fontsource on first use and cached (Electron: user data folder / browser: localStorage).

How to register

  1. Find fonts… — Search and filter (Japanese / category) in the modal → select → add to library (by default also adds to the current project)
  2. Google Fonts URL — Paste https://fonts.googleapis.com/css2?family=... directly (multiple families in one URL are registered together)
  3. From library + — Add previously registered fonts to another project

Two storage layers

LayerStorageScope
Library Electron: userData/fonts-library.json / browser: localStorage App-wide (reuse across projects)
Project fonts[] Included in project JSON That project only (Undo target, export target)

URL registration also saves automatically to the library. × on a library entry removes it from the library only; fonts already in a project remain.

Weight (Regular / Bold)

On registration, Fontsource resolves TTF URLs for 400 (Regular) and 700 (Bold) (Regular only if 700 is unavailable). Setting text weight to Bold uses the Bold TTF for outline conversion and native preview as well.

Main fields per registered entry:

  • family — Font name (e.g. Roboto)
  • cssUrl — Google Fonts CSS for display (wght@400;700 when Bold is supported)
  • fileUrl — TTF for outline (400)
  • fileUrlBold — TTF for outline (700, optional)
Usage example

Add Roboto to the library once in the modal, then use library + in the next project. To share with a team, pass the project JSON fonts[] as-is to reproduce the same typeface setup.

Text outline conversion

Converts text into path shapes per glyph. Use when you want logo or engraved text as printable 3D outlines.

  1. Select one text shape
  2. Design tab → Outline, or right-click → Outline
  3. The original text is replaced by a group (collection of paths)
Desktop only

Outline conversion is available only in the Electron app (Core Text on macOS, fontkit / HarfBuzz elsewhere). The button is disabled in a browser-only setup. Registered fonts download TTF before outlining.

See Editing — Text outline conversion for detailed editing steps.

Dimensions (D)

Draw horizontal and vertical dimensions to show lengths on the drawing. Does not affect 3D generation.

  1. Select the dimension tool (D)
  2. Click two points to measure in order
  3. A third click sets the dimension line position (offset)
  4. Values are computed from coordinates; manual override is available in the Design tab

Arrow style (dot, arrow, slash, etc.), text size, and prefix/suffix (e.g. L=, mm) can also be set.

Canvas images

Use the toolbar Place image button to put PNG / JPEG / WebP / SVG files directly on the drawing canvas. Placed images are normal objects: select, move, resize, rotate, flip, and save them in project JSON.

  • Formats — PNG / JPEG / WebP / SVG
  • Editing — In the Design tab, edit X/Y, width, height, opacity, and inspect stored image size
  • Export — Included in SVG / PDF output
  • Storage — Raster images are downscaled to a max side of 1600 px on import and stored as WebP only when that reduces size
Different from reference images

A canvas image is a normal object on a layer. A reference image is a page underlay in the Pages tab, used for scale calibration and tracing.

Sketch import (reference image)

Place a hand-drawn sketch or photo as a reference underlay, calibrate scale, then trace outlines. With AI integration (MCP), an external Vision model can place ghost shapes (semi-transparent proposals) for you to review and confirm.

1. Load a reference image

  1. Open the right panel Pages tab
  2. Expand the Reference image section
  3. Click Load image and choose a PNG / JPEG / WebP file
  4. Adjust Opacity so the underlay is easy to trace over

After loading, clicking the canvas does not move the underlay. Use the steps below only when you need to adjust position or size.

2. Move and resize (optional)

When you need to adjust placement or on-screen size, edit with the same frame and handles as rectangle selection.

  1. Click Move & resize (blue frame and eight handles appear)
  2. With Select (V) or Hand (H) still active, drag to move or drag handles to resize (Shift locks aspect ratio)
  3. When finished, click the highlighted Move & resize button again to leave edit mode
  • The button is highlighted in blue while editing
  • Until you exit edit mode, clicking the canvas alone does not start another edit (avoids accidental moves)
  • Esc also exits edit mode
  • Switching to another drawing tool (rectangle, line, etc.) exits automatically
  • Each drag is committed on mouse up and can be undone with Ctrl+Z

3. Calibrate scale

Pick one line of known real length on the underlay (e.g. a dimensioned edge or a measured span) to scale the entire image to real units.

  1. Click Calibrate scale (any active move/resize edit mode ends automatically)
  2. On the reference image, click one end of that line
  3. Click the other end of the same line (blue/red markers and a dashed line appear)
  4. Enter the real distance (mm) between those points and click Apply in the scale calibration panel
  5. Press Esc or Cancel in the panel to abort
Accuracy tip

Scale calibration sets the drawing’s overall accuracy. Use a dimension written on the sketch or a edge you measured. Hand sketches are approximate — expect to refine with the rectangle tool and dimensions after calibration.

4. Ghost shapes (AI proposals)

When AI uses MCP digitize_sketch, rectangles, circles, and lines appear as dashed, semi-transparent ghosts. You can edit or delete them, but they are excluded from 3D generation until confirmed.

  • Confirm ghosts — convert to normal shapes (3D profiles enabled from this point)
  • Remove ghosts — discard all proposals

For manual tracing, draw outlines with the rectangle (R) or pen tools as usual. See also AI / MCP — Sketch digitize.

Pages and layers

A project has multiple pages (top view, front view, etc.), each with layers.

  • Pages — Add and switch in the Pages tab. Choose the face direction when adding top/front/side pages for 3D
  • Layers — Manage in the Layers tab: show/hide, lock, rename
  • Newly drawn shapes are always added to the currently selected layer

Snap

While drawing, the cursor auto-snaps to nearby endpoints, intersections, midpoints, and more. Toggle in the Pages tab with Snap.

TypeMeaningIndicator
EndpointShape corners, line ends, bezier nodes□
IntersectionWhere lines cross×
MidpointEdge center△
CenterCircle or rectangle center◎
PerpendicularFoot of perpendicular from cursor to segment⊥

Priority: endpoint → intersection → midpoint → center → perpendicular. If none match, snap to the grid (1 / 5 / 10 mm).

Alignment, boolean ops, and more

For multi-shape alignment, grouping, union/subtract, and Undo/Redo, see Editing.

Rotation and flip

Reflected in outlines

Shape rotation, horizontal flip, and vertical flip are reflected in display, BBox, alignment, Profile extraction, and 3D generation. Dimension annotations do not automatically track edited shapes, so redraw dimensions when needed.

Units and scale

Millrect values are mm-based (internally 1 mm = 10 paper units). Page scale (e.g. 1/10) determines the relationship between paper distance and real size.

Check the current page scale in the status bar SCALE display. Use scale 1/1 for drawings intended for 3D generation.
