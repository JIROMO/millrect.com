Download

To build from source, see the GitHub repository.

Install on macOS

  1. Open the DMG — pick Apple Silicon or Intel for your Mac.
  2. Drag Millrect to Applications
  3. First launch — you may see an unsigned-app warning (see below).

Security warnings

Unsigned pre-release distribution

We do not code-sign or notarize with Apple yet. Download only from GitHub Releases.

If macOS says the app cannot be opened because the developer cannot be verified:

  1. Right-click (or Control-click) Millrect.app in Finder
  2. Choose Open, then confirm Open again

Or use System Settings → Privacy & Security → Open Anyway when offered. Later launches work normally.
