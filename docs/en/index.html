Millrect User Guide

From drawing to solid.

Millrect generates 3D models from orthographic views such as top and front projections. Available in the browser and as a desktop app. The guide walks through the real Module Joint 1 product: 24×100 mm, 2 mm thick, documented at 1:1 on A4 portrait.

Getting Started → Design Atlas Desktop app 3D workflow

Drawings come first

3D can always be regenerated from drawings. Edit the drawing and the solid updates.

Multiview solidification

Combine two or more views (e.g. top + front) to build 3D via CSG intersection.

Autosave

Work is saved automatically. Resume where you left off on the next launch.

STL export

Export the model from the 3D preview as an STL file.
