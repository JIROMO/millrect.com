デスクトップ版のダウンロード

Millrect はブラウザ版（millrect.com/app）でも使えますが、 AI 連携・テキストの 3D 輪郭化などはmacOS デスクトップ版向けの機能です。 プロジェクトの JSON 出力・インポートはブラウザ版でも利用できます。

現状 macOS のみ配布

Windows 版は未リリースです。Windows では ブラウザ版をご利用ください。

ダウンロード

最新版は GitHub Releases から入手できます（MIT ライセンス・無料・macOS のみ）。

ソースからビルドする場合は GitHub リポジトリ を参照してください。

macOS のインストール

  1. DMG を開く — Apple Silicon か Intel か、お使いの Mac に合わせて選びます。
  2. Millrect を Applications にドラッグ
  3. 初回起動 — 未署名配布のため警告が出る場合があります（次の節を参照）。

セキュリティ警告について

現状は未署名のプレリリース配布です

Apple のコード署名・公証は行っていません。配布ファイルは GitHub の Releases から直接入手してください。

「開発元を確認できないため開けません」と表示された場合:

  1. Finder で Millrect.app を右クリック（または Control+クリック）
  2. 開く を選び、ダイアログで再度 開く

または システム設定 → プライバシーとセキュリティ に「このまま開く」が表示されたらそこから許可できます。 2 回目以降は通常どおり起動できます。
