保存と出力

プロジェクトの保存・読み込み、図面の PDF / SVG 出力、3D の STL 出力、自動保存について説明します。

自動保存

図形を追加・編集するたびに、作業内容は自動的に保存されます。 ステータスバー右端の「保存済」と時刻で、最後に保存されたタイミングを確認できます。

アプリを終了して再度起動すると、前回の作業を再開するかどうかを選べます。 別の場所にファイルとして残したい場合は、下記の「プロジェクトファイルへの保存」を使ってください。

プロジェクトファイル（JSON）

Millrect のプロジェクト全体（全ページ・レイヤー・図形・寸法線・登録フォント・キャンバス画像・設定）を 1 つのファイルに保存します。

プロジェクト JSON の fonts[] には、Google Fonts 登録情報（family / cssUrl / fileUrl / fileUrlBold 等）が含まれます。 ライブラリ（fonts-library.json）はプロジェクトファイルとは別に、アプリのユーザーデータとして保持されます。 フォントの登録方法は 2D 描画 — プロジェクトフォント を参照してください。

操作方法
保存ツールバーの保存アイコン / メニュー「ファイル → 保存」/ Ctrl+S（Cmd+S）
開くツールバーのフォルダアイコン / メニュー「ファイル → 開く」
新規メニュー「ファイル → 新規」/ Ctrl+N

保存先フォルダとファイル名は、保存ダイアログで指定します。

SVG 出力・インポート

  • SVG 出力 — 現在表示中のページを SVG ファイルとして書き出します（Illustrator 等への受け渡し向け）
  • SVG インポート — 外部 SVG を現在のページのレイヤーに取り込みます

SVG は 2D 図面用です。キャンバス画像は含まれますが、3D 情報は含まれません。

PDF 出力

プロジェクト内の全ページを 1 つの PDF ファイルにまとめて出力します。キャンバス画像も含まれるため、印刷や共有に便利です。

メニュー「ファイル → PDF として書き出し」、またはツールバーの PDF アイコンから実行できます。

STL 出力（3D）

3D 生成 の手順で立体が表示されている状態で、 3D パネルの「STL」ボタンを押すと、3D プリント用の STL ファイルを保存できます。

STL 出力の前提

上面図 + 正面図など、2 軸以上のページが揃い、3D プレビューにメッシュが表示されている必要があります。
